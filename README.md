## Tech Stack

Next.js, React, TypeScript, Tailwind CSS, GSAP

## Featured Projects

- **AI Fake News Detector** — AI-powered fact-checking web app using Groq LLMs, Gemini Embeddings, and semantic search
- **DermaCam** — AI dermatology app integrating six ML models with Flutter and Flask
- **PoseMuse** — Real-time multi-person pose detection Flutter app using MediaPipe and Google ML Kit

## Running Locally

```bash
git clone https://github.com/abhizz-b/portfolio.git
cd portfolio
pnpm install
pnpm dev
```
